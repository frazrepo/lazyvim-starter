# Quick Install

Backup
```
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```
Clone this repo
```
git clone https://github.com/frazrepo/lazyvim-starter ~/.config/nvim
```

Start Neovim
```
nvim
```
